## Description
Componente para usos sarlanga.

## Basic use

```javascript
import Nav from 'avantrip-react/avantrip/Nav';

export default () => (
  <Nav
    onEvent={function}
    message={string}
    isActive={boolean}
  />
)
```


## Props

#### `onEvent={<Boolean> || <Function>}`
Event handler

#### `isActive={<Boolean>}`
Enable an action within the component
