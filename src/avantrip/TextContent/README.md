### Description
Componente para la home de la sección inferior (Avantrip es una agencia...) 

### Basic use

```javascript
import TextContent from 'avantrip-react/avantrip/TextContent';

export default () => (
  <TextContent
  	title={string}
  	subtitle={string}
  />
)
```

### Props

#### `title={<String>}`
Título

#### `subtitle={<String>}`
Subtítulo o texto de soporte
