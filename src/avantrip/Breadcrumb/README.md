## Description
Organiza los childrens y los separa utilizando el
styled-component **BreadcrumbSeparator**. Cada children es
envuelto en el componente **BreadcrumbItem**.

## Basic use

```javascript
import Breadcrumb from 'avantrip-react/avantrip/Breadcrumb';

export default () => (
  <Breadcrumb>
    {Node}
  </Breadcrumb>
)
```
