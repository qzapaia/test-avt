## Description
Componente para usos sarlanga.

## Basic use

```javascript
import ProductCard from 'avantrip-react/avantrip/ProductCard';

export default () => (
  <ProductCard
    onEvent={function}
    message={string}
    isActive={boolean}
  />
)
```


## Props

#### `onEvent={<Boolean> || <Function>}`
Event handler

#### `isActive={<Boolean>}`
Enable an action within the component
