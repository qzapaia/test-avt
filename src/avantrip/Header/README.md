## Description
Header de Avantrip que funciona como contendor.

## Basic use

```javascript
import Header from 'avantrip-react/avantrip/Header';

export default () => (
  <Header>
    <node/>
    <node/>
    <node/>
  </Header>
)
```
## Props

#### `phoneText={<String>}`
Es el número de teléfono de compra.
