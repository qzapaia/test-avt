## Description
Componente para usos sarlanga.

## Basic use

```javascript
import DestinationsListByPoints from 'avantrip-react/quiero/DestinationsListByPoints';

export default () => (
  <DestinationsListByPoints
    onEvent={function}
    message={string}
    isActive={boolean}
  />
)
```


## Props

#### `onEvent={<Boolean> || <Function>}`
Event handler

#### `isActive={<Boolean>}`
Enable an action within the component
