## Description
Componente para usos sarlanga.

## Basic use

```javascript
import JustOne from 'avantrip-react/global/JustOne';

export default () => (
  <JustOne
    onEvent={function}
    message={string}
    isActive={boolean}
  />
)
```


## Props

#### `onEvent={<Boolean> || <Function>}`
Event handler

#### `isActive={<Boolean>}`
Enable an action within the component
