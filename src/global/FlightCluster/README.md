## Description
Componente cluster para el resultado de búsqueda

## Basic use

```javascript
import FlightCluster from 'avantrip-react/global/FlightCluster';

export default () => (
  <FlightCluster
    onClick={function}
    data={object}
  />
)
```

## Props

#### `onClick={<Function>}`
Event handler

#### `data={<Object>}`
Información que necesita.
